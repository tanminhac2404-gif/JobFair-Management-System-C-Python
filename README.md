# JobFair-Management-System-C-Python
Sample code for C-Python application
Project Title: Hệ thống Quản lý Jobfair

Tech Stack: C (Logic & Knapsack Algorithm), Python (Tkinter GUI), Ctypes (Interoperability).  

Key Features:

Quản lý gian hàng thực tế (1-30).  

Thuật toán tối ưu hóa Knapsack để chọn doanh nghiệp có giá trị cao nhất dựa trên tải trọng sàn.  

Chức năng Chỉnh sửa (Alter) và Xóa (Delete) linh hoạt.  

How to run: chạy file exe hoặc điều chỉnh source code theo ý muốn.
